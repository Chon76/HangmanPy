## SETUP:

### Server

```
cd root/backend/
```

```
sudo pip install virtualenv
python3 -m venv venv
```

_On Mac_

```
. venv/bin/activate
pip install -r requirements.txt
flask run
```

_On Windows_

```
venv\Scripts\activate
pip install -r requirements.txt
flask run
```

### Client

```
cd root/frontend/
```

```
npm install
npm start
```
